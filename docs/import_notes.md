# Заметки по подготовке импортных данных

- База данных нормализована до отдельных справочников ролей, категорий, единиц измерения, поставщиков, производителей, статусов заказов и пунктов выдачи.
- В заказах найден артикул `O43COU8`, которого нет в исходном файле `Tovar.xlsx`. Для сохранения ссылочной целостности добавлена карточка товара-заглушки с изображением `picture.png`.
- В строке заказа встречалась невозможная дата `30.02.2025`; при импорте она заменена на `2025-02-28`.
- Файл `src/StroyMaterials.App/Data/stroymaterials.db` можно использовать как переносимую локальную БД. Если он удален, приложение создаст новую БД из `Database/create_schema.sql` и `Database/seed_data.sql`.
